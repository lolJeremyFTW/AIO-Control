## Free -- EUR 0

- 1 workspace
- 3 businesses
- Manual plus webhook triggers (geen cron quota)
- OpenClaw plus Hermes via uw eigen VPS
- Geen routine quota
- Community support

## Pro -- EUR 29 / maand

- Onbeperkt businesses
- Cron schedules (op subscription OF API key)
- Telegram plus email notificaties
- Spend limits plus auto-pause
- Mobile push (web plus Capacitor)
- **10.000 runs / maand inbegrepen**, daarna EUR 0,003 per run
- Email support

## Team -- EUR 99 / maand

- Alles uit Pro
- Onbeperkt members plus role-based access
- Per-business isolated mode (geen workspace fallback)
- Audit log export plus GDPR DSR helpers
- **100.000 runs / maand inbegrepen**, daarna EUR 0,002 per run
- Priority support (24u response)

## Yearly billing

10x maandprijs voor 12 maanden = 2 maanden gratis. Pro EUR 290/jaar, Team EUR 990/jaar.

## Wisselen van plan

Settings > Subscription > Change plan. Pro-rated billing via Stripe. Downgrade gaat in op de volgende billing cycle.
